HospiOne Website – React + Vite

This project is a hospital website built using React and Vite.
It includes a navigation bar, header, specialities section, workflow steps, and a footer, all styled with custom CSS.

Tech Used

React
Vite
CSS
Main Files

Navbar.jsx – Contains the full homepage layout

Navbar.css – Styling for all sections

Images stored in src/assets/

Run Project:
npm install
npm run dev
or
npx vite
